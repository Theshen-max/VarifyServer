# VarifyServer

**基于 Node.js 的安全验证微服务**

## 项目目标
利用 Node.js 优秀的异步事件驱动特性，快速处理验证码的生成、存储以及邮件投递业务。

## 技术栈
- **运行时**: Node.js
- **协议**: gRPC (Server 端)
- **中间件**: ioredis (管理验证码过期逻辑)
- **集成**: Nodemailer (对接 SMTP 邮件网关)

## 核心特性
- **时效管理**: 强依赖 Redis TTL，验证码 10 分钟自动过期，无需人工维护状态。
- **健壮性**: 完善的错误捕获机制，确保 SMTP 网络波动不影响 gRPC 服务的稳定性。
- **配置安全**: 邮箱授权码等敏感信息通过 `.env` 进行隔离管理。
